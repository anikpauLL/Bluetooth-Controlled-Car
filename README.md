# Bluetooth-Controlled-Car


A **Bluetooth-controlled Arduino car** designed and implemented as part of **ETE-300** coursework at **Chittagong University of Engineering and Technology (CUET)**. This project demonstrates the practical use of **Bluetooth communication**, **Arduino microcontroller programming**, and **DC motor control** to build a remotely operated mobile vehicle.

---

##  Project Overview

This project focuses on building a simple, reliable, and educational **remote-controlled car** using Bluetooth technology. A smartphone application communicates wirelessly with an Arduino via an **HC-05 Bluetooth module**, allowing the user to control the carâ€™s movement in real time.

The system does not use sensors, making it a **manual control vehicle** rather than an autonomous robot. It serves as an excellent learning platform for beginners in embedded systems and robotics.

---

##  Aims & Objectives

---

## Working Principle

1. A smartphone sends control commands via Bluetooth
2. The **HC-05 Bluetooth module** receives the commands
3. Arduino processes the received data
4. Motor driver (L293D) controls the DC motors
5. The car moves forward, backward, left, or right based on user input

## How to Run the Project

1. Assemble the car chassis and mount motors
2. Connect motors to the L293D motor driver
3. Interface HC-05 Bluetooth module with Arduino
4. Upload the Arduino code using Arduino IDE
5. Power the system using a 12V battery
6. Pair smartphone with HC-05 module
7. Control the car using the Bluetooth RC app

## Limitations

* Limited Bluetooth communication range
* Requires sufficient power supply
* Performance depends on motor torque
* Sensitive to weight and wheel alignment
* No autonomous or sensor-based control
